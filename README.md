# NOICE - The Digital Void 🌌

動画から「存在（背景）」を消去し、「意思（動き）」だけを抽出するアート・ビデオプロセッサー。

## Features
- **Background Subtraction**: 動きのある部分だけをノイズで可視化。
- **Real-time Streaming**: ASGIサーバーによる低遅延プレビュー。
- **Audio Synthesis**: オリジナル音声の結合、またはホワイト/ブラウンノイズの生成。
- **High-End Design**: 虚無をテーマにしたプレミアムなUI/UX。

## Tech Stack
- Python (FastAPI, OpenCV, NumPy, MoviePy)
- Vanilla JavaScript & CSS (Glassmorphism)

## Requirements
- Python 3.10+
- OS: Windows 10/11 recommended

## Setup
1. `run_noice.bat` を実行（初回は自動的に依存関係をインストールします）。
2. ブラウザで `http://127.0.0.1:8000` にアクセス。
